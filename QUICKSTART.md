# 🚀 빠른 시작 가이드

## 3분 안에 실행하기

### 1단계: 환경변수 설정

루트 디렉토리에서 `.env.example`을 `.env`로 복사:

```bash
# 프로젝트 루트에서
cp .env.example .env
```

`.env` 파일을 열어 OpenAI API 키를 입력:

```env
# 필수
OPENAI_API_KEY=sk-your-actual-openai-key-here

# 선택 (trend_agent, competitor_agent용 - 없으면 모의 데이터 사용)
NAVER_CLIENT_ID=your_naver_client_id
NAVER_CLIENT_SECRET=your_naver_client_secret
```

**중요:**
- `OPENAI_API_KEY`는 필수입니다.
- Naver API 키가 없어도 모든 에이전트가 작동합니다 (모의 데이터 사용).

---

### 2단계: 백엔드 실행

```bash
# 터미널 1
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

✅ 서버가 `http://localhost:8000`에서 시작됩니다.

**백엔드 실행 명령어 설명:**
- `uvicorn`: FastAPI의 공식 ASGI 서버
- `app.main:app`: `app/main.py` 모듈의 `app` 객체 실행
- `--reload`: 코드 변경 시 자동 재시작 (개발 모드)

---

### 3단계: 프론트엔드 실행

```bash
# 터미널 2 (새 터미널)
cd frontend
npm install
npm run dev
```

✅ 브라우저에서 `http://localhost:5173`으로 자동 이동됩니다.

---

### 4단계: 테스트

채팅창에 다음 중 하나를 입력:

**세그먼트 분류:**
```
에어팟 프로 구매자를 세그먼트로 분류해줘
```

**트렌드 분석 (PDF 다운로드):**
```
스마트워치 최근 3개월 트렌드 알려줘
```

**광고 문구 생성 (멀티턴 지원):**
```
친환경 세제 광고 문구 만들어줘
```
→ 다음 메시지: `더 만들어줘`

**경쟁사 분석 (HTML 보고서):**
```
아이폰 15와 갤럭시 S24 비교 분석해줘
```

**🆕 마케팅 전략 종합 보고서:**
```
지금까지의 결과를 종합해서 마케팅 전략 보고서 만들어줘
```

에이전트가 자동으로 태스크를 감지하고 실행합니다!

## 🎯 활성화된 에이전트 (6개)

### ✅ 1. 사용자 세그먼트 분류
**키워드**: 세그먼트, 고객분류, 타겟, 페르소나, 클러스터, 그룹

**예시**:
- "에어팟 프로 구매자를 세그먼트로 분류해줘"
- "고객 데이터를 세그먼트로 나눠줘"

**기능**: K-means 클러스터링, PDF 리포트 생성

---

### ✅ 2. 소비 트렌드 분석
**키워드**: 트렌드, 유행, 인기, 검색량, 관심도, 소비

**예시**:
- "스마트워치 최근 3개월 트렌드 알려줘"
- "반려동물 사료 6개월 트렌드 분석해줘"

**기능**:
- Naver DataLab API 연동 (환경변수 없으면 모의 데이터)
- 연관 키워드 클러스터 분석
- PDF 리포트 다운로드

---

### ✅ 3. 광고 문구 생성
**키워드**: 광고, 문구, 카피, 헤드라인, 슬로건, 마케팅

**예시**:
- "친환경 세제 광고 문구 만들어줘"
- "더 만들어줘" (멀티턴 대화)

**기능**:
- LLM 기반 광고 문구 생성
- 컴플라이언스 검사 (금지어 체크)
- RAG 활용
- **멀티턴 대화 지원**

---

### ✅ 4. 경쟁사 분석
**키워드**: 경쟁사, 비교, 가격, 시장, 벤치마크, 경쟁

**예시**:
- "아이폰 15와 갤럭시 S24 비교 분석해줘"
- "에어팟 프로 경쟁사 분석해줘"

**기능**:
- Naver Shopping API 연동 (환경변수 없으면 모의 데이터)
- SWOT 분석
- Chart.js 시각화 (레이더 차트, 파이 차트)
- HTML 보고서 다운로드

---

### ✅ 5. 리뷰 감성 분석
**키워드**: 리뷰, 감성, 평가, 후기, 댓글, 의견

**예시**:
- "아이폰16 리뷰 분석해줘"
- "에어팟 프로 구매자들의 리뷰 감성 분석을 진행해줘"

**기능**:
- 웹 크롤링을 통한 실제 리뷰 데이터 수집
- LLM 기반 감성 분석
- LDA 토픽 모델링
- PDF 리포트 다운로드

---

### ✅ 6. 🆕 마케팅 전략 종합 보고서
**키워드**: 종합, 통합, 전체, 보고서, 정리, 마케팅 전략, 전략 보고서

**예시**:
- "지금까지의 결과를 종합해서 마케팅 전략 보고서 만들어줘"
- "전체 분석 결과를 통합해서 정리해줘"

**기능**:
- 한 세션 내에서 실행된 모든 태스크 결과를 DB에서 조회
- LLM이 모든 결과를 종합하여 통합 마케팅 전략 생성
- Executive Summary, 시장 환경 분석, 고객 인사이트, 실행 계획 제시

**사용 시나리오**:
```
1. 에어팟 프로 트렌드 분석해줘
2. 에어팟 프로 광고 문구 만들어줘
3. 에어팟 프로 구매자를 세그먼트로 분류해줘
4. 에어팟 프로 리뷰 분석해줘
5. 에어팟 프로 경쟁사 분석해줘
6. 지금까지의 결과를 종합해서 최종 전략 보고서 만들어줘 ← 🆕
```

---

## 🛠️ 문제 해결

### 백엔드 오류

**오류:** `ModuleNotFoundError`
**해결:** 가상환경 활성화 후 `pip install -r requirements.txt` 재실행

**오류:** `OpenAI API 키가 설정되지 않았습니다`
**해결:** `backend/.env` 파일에서 `OPENAI_API_KEY` 확인

### 프론트엔드 오류

**오류:** `Cannot connect to backend`
**해결:** 백엔드가 8000 포트에서 실행 중인지 확인

**오류:** CORS 오류
**해결:** 백엔드 재시작
